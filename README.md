# Campus-Course-Record-Manager
Think of this project as a smart, digital logbook for a college, built entirely in Java. Instead of a complicated web app, it runs right in your computer's terminal (command line), making it super fast and lightweight.

The breakdown of what it actually does given below:

 - Keeps Track of People and Classes: It lets an admin easily type in new students, create college courses, and assign teachers to those classes.

 - Handles the Math and Rules: When a semester ends, you can enter a student's grades, and the app automatically calculates their GPA. It also acts like a bouncer, stopping students from enrolling in too many credits at once.

 - Saves Data Simply: Instead of hooking up to a massive database like MySQL, it just saves all the student and course info into basic CSV files (which open like Excel spreadsheets). When you close the app, your data is safely tucked away in those files.

 - Auto-Backups: It automatically makes copies of your data and saves them in folders stamped with the current date and time, so you never accidentally lose a student's record.

In short, it's a clean, simple Java program that uses core coding concepts—like Object-Oriented Programming and file handling—to automate the boring paperwork of running a school.
